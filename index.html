About

Internal tool · Python / Flask

Time Exception Slip Automation

Problem: ~128 transit employees required hand-filled Time Exception Slips every pay period — names, IDs, OT hours, Excel summaries. Four-plus hours of repetitive work with real risk of typos and payroll mismatches.

What I built: A browser-based tool that generates pre-filled slip PDFs, handles biweekly OT entry with live validation, and exports a formatted Excel summary for Finance — all in one workflow. Data stays local; no server storage required.

Internal automation · Microsoft 365

Missing Time Exception Slip — Automated Outreach

Problem: Chasing missing time exception slips meant back-and-forth messages, unclear ownership, and risk to payroll accuracy, audit trails, and compliance.

What I built: A low-code flow using SharePoint, Microsoft Lists, Power Automate, and Outlook. When a slip is missing, the operations manager receives a structured email with employee name, date out, leave bank, and time — with key transit administration teammates CC'd for visibility. Minimal custom code; relies on M365 automation the department already owns.
