# David Godinez — Portfolio

Minimal portfolio site for public sector program analysis and operational tooling.

## Local preview

```bash
python3 -m http.server 8080
# Open http://localhost:8080
```

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Settings → Pages → Source: `main` branch, `/ (root)`
3. Site will be live at `https://davidgo24.github.io/david-godinez-portfolio/`

## Deploy to Vercel

Import the repo in [Vercel](https://vercel.com) — no build step required (static HTML).
